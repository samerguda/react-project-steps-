خطوات عمل مشروع في React

1. إعداد بيئة العمل

تأكد من أن لديك Node.js و npm مثبتين على جهازك. يمكنك التحقق من التثبيت باستخدام الأوامر التالية في سطر الأوامر:

node -v
npm -v

قم بتثبيت أداة إنشاء تطبيق React باستخدام npm:

npm install -g create-react-app

2. إنشاء مشروع React جديد

استخدم create-react-app لإنشاء مشروع جديد:

npx create-react-app my-app

قم بتغيير الدليل إلى مجلد المشروع الجديد:

cd my-app

3. تشغيل التطبيق

قم بتشغيل التطبيق الافتراضي للتأكد من أن كل شيء يعمل بشكل صحيح:

npm start

سيفتح المتصفح تلقائيًا على http://localhost:3000 لعرض التطبيق.

4. بنية المجلدات والملفات

داخل مجلد المشروع، ستجد بنية الملفات الأساسية:

my-app/
├── node_modules/
├── public/
├── src/
├── .gitignore
├── package.json
├── README.md

5. تطوير التطبيق

src/index.js: هو نقطة الدخول للتطبيق.

src/App.js: هو المكون الرئيسي للتطبيق.

يمكنك تعديل هذه الملفات وإضافة مكونات جديدة داخل مجلد src لتطوير واجهات المستخدم.

6. إضافة المكونات

لإنشاء مكون جديد، يمكنك إنشاء ملف جديد في مجلد src، على سبيل المثال src/components/MyComponent.js:

import React from 'react';

function MyComponent() {
  return (
    

      
Hello, World!

    

  );
}

export default MyComponent;

قم بإضافة هذا المكون إلى App.js:

import React from 'react';
import MyComponent from './components/MyComponent';

function App() {
  return (
    

      
    

  );
}

export default App;

7. إدارة الحالة (State) والتفاعل مع المستخدم

يمكنك استخدام useState لإدارة الحالة المحلية داخل المكونات.

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    

      
You clicked {count} times

      
    

  );
}

export default Counter;

8. التعامل مع البيانات

يمكنك استخدام fetch أو أي مكتبة أخرى مثل Axios لجلب البيانات من API.

import React, { useEffect, useState } from 'react';

function DataFetching() {
  const [data, setData] = useState([]);

  useEffect(() => {
    fetch('https://api.example.com/data')
      .then(response => response.json())
      .then(data => setData(data));
  }, []);

  return (
    

      {data.map(item => (
        
{item.name}

      ))}
    

  );
}

export default DataFetching;
